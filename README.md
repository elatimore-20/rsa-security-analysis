# RSA Security Analysis Project

This project implements RSA from scratch in Python and evaluates three classical attacks:
- Broadcast attack
- Fermat factorization
- Common modulus attack

## Files
- `rsa.py`: RSA key generation, encryption, and decryption
- `attacks.py`: attack implementations
- `experiments.py`: experiment scripts and graph generation
- `app.py`: Streamlit demo
- `main.py`: runs demonstrations

## Run
```bash
python experiments.py
streamlit run app.py
